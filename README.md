# Akıllı Kilit Sistemi

PIC16F877A tabanlı şifreli akıllı kilit sistemi projesi.

Bu proje, gömülü sistemler ve PCB tasarımı alanında kendimi geliştirmek amacıyla hazırlanmıştır.
Proje kapsamında PIC mikrodenetleyici programlama, keypad kullanımı, LCD kontrolü, Proteus devre tasarımı ve
PCB oluşturma konularında deneyim kazanılması hedeflenmiştir.

## Özellikler
- 4x3 keypad ile şifre girişi
- LCD ekran desteği
- Doğru/yanlış şifre kontrolü
- LED durum göstergeleri
- Buzzer uyarı sistemi

## Kullanılan Malzemeler
- PIC16F877A
- 16x2 LCD
- 4x3 Keypad
- LED
- Buzzer
- Dirençler

## Yazılım
- MPLAB X IDE
- XC8 Compiler
- Proteus 8 Professional

## Proje İçeriği
- C kaynak kodu
- HEX dosyası
- Proteus şematik tasarımı
- PCB tasarımı
- 3D PCB görüntüsü
